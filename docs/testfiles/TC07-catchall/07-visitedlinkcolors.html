14. Color (hue) only to identify link, indicate visited/unvisited, and identify active/hover

The blue (unvisited), purple (visited), green (hover), and orange (active) link text all have similar luminance/lightness. All have sufficient contrast between background/foreground, but insufficient contrast between visited, unvisited, hover, and active colors. All also have insufficient contrast between the link text and surrounding text to differentiate the links from the text. The cursor is also the same for links as it is for text, so does not provide a method to differentiate from surrounding text.

27. Color (hue) AND some other method to identify link, indicate visited/unvisited, and identify active/hover

27.a. COLOR (hue) AND CONTRAST to identify and differentiate between visited/unvisited, active and hover.

Turns out that it is possible to use contrast to differentiate between a link and it's surrounding text while sill maintaining sufficient contrast with the background. HOWEVER, it is NOT possible to use contrast to distinguish between a visited link, unvisited link, AND surrounding text while ALSO maintaining sufficient contrast with the background. Some other method must be used in addition to contrast (e.g., text decoration, weight, outline, etc.).

  • Unvisited link (blue) - contrast sufficient with background, but NOT sufficient to distinguish from surrounding text. Text underline used to distinguish from surrounding text.
    • Hover link (green) - same contrast as unvisited link; dotted underline to differentiate between unvisited, visited, and hover
    • Active link (orange) - same contrast as univisited link; wavy underline to differentiate between univisited, visited, hover, and active
  • Visited link (purple, underlined) - contrast sufficient with background AND to distinguish from unvisited link.
