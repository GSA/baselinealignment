Frame 1

The frame element has been removed in HTML5. There is no replacement element for this. Other elements that have been removed are the frameset and the noframes elements

Another test page is available to to test Frames developed in HTML4.01.

Frame 2

The iframe element is still a valid element in HTML4.01. It allows you to display a web page within a web page. To meet accessibility requirements, the iframe must have a title or name that is descriptive.
